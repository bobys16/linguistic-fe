# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-09

### Added
- Initial release of Digital Psycholinguistics Workbook Frontend
- User authentication (login, register, JWT)
- Dashboard with progress tracking
- Module system with 4 core modules:
  - Working Memory
  - Processing Speed
  - Noticing
  - Reflective Practice
- Task runner with 5 task types:
  - Working Memory Span Task
  - Lexical Decision Task
  - Sentence Verification Task
  - Error Correction Task
  - Reflection Prompt Task
- Learning journal with tagging system
- Progress analytics and statistics
- Mobile-first responsive design
- Academic light theme UI
- React Query for data fetching and caching
- Protected routes with middleware
- Comprehensive TypeScript types
- Production-ready configuration

### Technical Stack
- Next.js 14+ with App Router
- TypeScript
- Tailwind CSS
- Axios for API communication
- React Query (TanStack Query)
- JWT authentication
- Context API for auth state

### Documentation
- Complete README with setup instructions
- Deployment guide for multiple platforms
- Contributing guidelines
- API integration documentation
